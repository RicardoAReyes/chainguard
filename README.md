# Chainguard Demo Projects

A collection of demo projects showcasing [Chainguard](https://chainguard.dev) hardened container images, OS packages, and JavaScript libraries for secure-by-default application stacks.

## Projects

| Project | Stack | Description |
|---|---|---|
| [wp-nginx-mariadb](./wp-nginx-mariadb) | WordPress · Nginx · MariaDB · Node | WordPress stack using Chainguard images with a custom JS theme build pipeline |

## Registries Used

| Registry | Purpose |
|---|---|
| `cgr.dev/chainguard-private` | Hardened container images |
| `packages.wolfi.dev` | Wolfi OS APK packages |
| `libraries.cgr.dev/javascript` | SLSA-attested JavaScript libraries |
